\subsection{Java}
Java ist eine allzweck Programmiersprache, welche Klassen- und Objektorientiert ist 
und sich vor allem durch ihre Plattformunabhängigkeit auszeichnet.  
Der Quellcode wird hierführ in bytecode kompiliert, der dann auf jeder Java virtual machine
unabhängig von System oder Architektur zum laufen gebracht werden kann. \cite{javawiki}
